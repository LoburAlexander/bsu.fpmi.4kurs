Tracks Points from 1.Image to 2.Image. Need initImgs before start and at the end of the program for cleanup.

Parameters
imgIprevious Image source. (isn't changed)
imgJactual Image target. (isn't changed)
ptsIpoints to track from first Image. Format [0] = x1, [1] = y1, [2] = x2 ...
nPtsInumber of Points to track from first Image
ptsJcontainer for calculated points of second Image. Must have the length of nPtsI.
nPtsJnumber of Points
levelPyramidlevel, default 5
fbOutforward-backward confidence value. (corresponds to euclidean distance between). Must have the length of nPtsI: nPtsI * sizeof(float).
nccOutnormCrossCorrelation values. needs as inputlength nPtsI * sizeof(float)
statusOutIndicates positive tracks. 1 = PosTrack 0 = NegTrack needs as inputlength nPtsI * sizeof(char)
Returns
1, if success; 0, if nPtsJ != nPtsI
