\subsection{Detailed Description}
The Base Class for 1\-D or Row-\/wise Filters

This is the base class for linear or non-\/linear filters that process 1\-D data. In particular, such filters are used for the \char`\"{}horizontal\char`\"{} filtering parts in separable filters.
