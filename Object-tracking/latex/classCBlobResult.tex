\subsection{Detailed Description}
Classe que contï¿½ un conjunt de blobs i permet extreure'n propietats o filtrar-\/los segons determinats criteris. Class to calculate the blobs of an image and calculate some properties on them. Also, the class provides functions to filter the blobs using some criteria.
